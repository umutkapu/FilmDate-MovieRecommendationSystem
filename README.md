FilmDate is a full-stack web application that helps couples find the perfect movie to watch together.

Unlike traditional recommendation systems, FilmDate generates suggestions based on two movies selected by different people, combining classic rule-based filtering with AI-powered semantic embeddings to deliver personalized and diverse results.

 **_Key Features_**

AI-based movie recommendations using sentence embeddings

Hybrid scoring (semantic similarity + genres + ratings)

FastAPI backend with intelligent caching for low latency

React frontend optimized for desktop and mobile

Deployed on Vercel (frontend) and Render (backend)

 **Tech Stack**

Frontend: React, Vite, CSS

Backend: FastAPI, Python

AI / ML: Hugging Face sentence embeddings (MiniLM)

Data: TMDB API

Deployment: Vercel, Render

FilmDate is designed as a real-world, production-ready project focusing on performance, scalability, and user experience.
